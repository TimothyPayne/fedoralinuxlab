What is Linux?

Linux is a free and open-source computer operating system, like Windows only much better and there are big fundamental differences. Fedora allows you to change the desktop you see not just colors, you can switch to a different lightweight desktop for older computers short on RAM.

What is Open Source?

Open Source allows you access to the source code (programming) so it is ‘Open’. Why give it away? There are countless reasons many make a version available free, and offer extra options for the paid version.

What is Fedora?

Red Hat (DoD spends a Billion+ with them) produces Fedora which is designed to serve as a public testing ground and technology incubator for new features.
