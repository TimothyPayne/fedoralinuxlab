Windows 10 Backup

Recovery Drive

This is needed to recover from a data disater, Microsoft is very strict so no one can install a free copy of Windows.

Type in search box in the Lower Left corner: Recovery Drive and select it on the top of the list.

Check: Backup System Files needs to be selected, this is the default and should already be checked.

Click Next to start - Please Wait

This takes time it can be confusing, it is making a small drive inside like putting a partition in a box.

Select the USB flash drive The available USB drive will be listed

Click: Next WARNING: All data on the drive will be deleted.

Click: Create - Creating the recovery drive - This can take a very long time depending on your USB drive.

Click: Finish - The recovery drive is ready

Partition the drive: Extream Caution must be used here, Measure twice - Cut once

In the Lower Left search box type: Partition - Select Create and Format Partitions

Disk Management will show the list of drives. The C: drive is your Windows the file system is NTFS

RECOVERY is the USB drive you are working with it has the FAT32 filesystem. Check twice.

Right Click: Unallocated section. It is blank and can not be written to yet.

Click: New Simple Volume at the top of the list.

Click: Next - In the New Simple Volume Wizard

Click: Next - The Maximum number at the top should match the box at the bottom - defualt, is the size of the drive.

Click: Next - The defualt letter Windows gave it, you can change it if you want.

Format the drive - Defualt is NTFS - Volume Lable to name the drive 'Backup' is a good choice

Click: Next - The check box for 'quick format' is the default

Click: Finish - The list is all you have done so far, double check the information.

Close all open windows you are finished, next is doing a Backup to your snappy new drive, good job.

Windows has changed Windows Backup to use cloud storage that they will happily sell you, or to use the USB drive:

In the Lower Left search box type: Control Panel.

Select Control Panel from the top of the list.

System and Security is on the top on the Left

Select Backup and Restore (Windows 7) at the bottom of that section 3rd one down above Network and Internet.

Back up or restore your files. The Change settings is at the bottom of the list to select drive it is above Restore

The pop up window - Double click to select the drive to backup to.

Set up back up: Let Windows choose is the defualt and recomended

Click: Next

Review your backup settings - Defaults work fine, you can change the time from Sunday

Click: Save settings and run backup.

Windows will run the backup, you can start the backup at any time you do not have to wait for the day of the week.

Backup in progress... Click the View Details if you want to watch the individual files zip by.
